# 📱 Attendee Companion

The VenueIQ mobile app provides fans with a personalized, high-tech experience to navigate and enjoy the venue.

## 📱 Experience

- **Indoor Pathfinding**: Turn-by-turn directions to seats, food, and exits.
- **Real-time Queue Status**: Intelligent estimates for concessions.
- **AR Wayfinding**: Augmented reality overlays for easy navigation (Experimental).

## 🛠 Tech Stack

- **Platform**: React + Vite (Optimized for mobile web views).
- **Styling**: Modern CSS with dark mode support.
- **Icons**: Lucide React
- **State**: React Hooks for dynamic navigation updates.

## 🚀 Getting Started

```bash
# Install dependencies
npm install
```

```bash
# Run in development mode
npm run dev
```

```bash
# Build for production
npm run build
```
